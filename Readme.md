# 文件规约

* 目录结构：

  * 课件和作业：课程名称/开课时间/授课老师/(课件、作业)/两位数字-章节名称

  * 往年卷：

    * 课程名称/试卷/(期中、期末)-(xxxx-xxxx学年第x学期)-(带答案、无答案、回忆版)

    * 课程名称/试卷/(期中、期末)-(xx级)-(带答案、无答案、回忆版)

## 提示词

````text
请按照“两位数字-章节名称”的格式对下面的文件名重新编号并命名，例如输入：
```
0-课程介绍-绪论-2025.pptx
1-2-2 栈：函数调用时的内存分配.pdf
3.机器学习导论-线性模型-2025.pdf
4_模式识别_Normalization_FLD_FaceRecog.pdf
```
输出：
```
01-课程介绍-绪论.pptx
02-栈：函数调用时的内存分配.pdf
03-线性模型.pdf
04-模式识别_Normalization_FLD_FaceRecog.pdf
```
请仅输出重命名后的结果（一行一个文件）。
````

````text
请尽量按照“(期中、期末)-(xxxx-xxxx学年第x学期)-(带答案、无答案、回忆版)”的格式重命名文件，例如输入：
```
14-16级微积分I期末试卷及参考答案.pdf
2023-2024学年第一学期线性代数期末试卷.pdf
2023-2024学年第一学期线性代数期末答案.pdf
2013年数据结构期末试卷（带答案）.pdf
```
输出：
```
期末-14-16级-带答案.pdf
期末-2023-2024学年第一学期-试卷.pdf
期末-2023-2024学年第一学期-答案.pdf
期末-2013-带答案.pdf
```
请仅输出重命名后的结果（一行一个文件）。
````
